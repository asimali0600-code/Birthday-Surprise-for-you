# 🌐 Deploy to GitHub Pages

### 1. Create Repository

Example:

```
birthday-for-afia
```

### 2. Upload Files

Upload everything to the repository.

### 3. Open

```
Settings
```

↓

```
Pages
```

↓

```
Deploy from Branch
```

↓

Select

```
main
```

↓

```
/(root)
```

Click **Save**.

Your website will be available at

```
https://YOUR_USERNAME.github.io/birthday-for-afia/
```
